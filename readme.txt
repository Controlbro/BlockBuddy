Step1 - Extract the folder and open it.
Step2 - open the command promt/terminal and run the command "npm install"
Step3 - Open the settings.json file and configure it.
Step3 - run the command "node ." or "npm start" to start the web server.
Step4 - Open your web browser and navigate to "http://localhost:3000" or the Domain to access.


Tip: You can use Vercel (http://vercel.com/), Netlify  (https://www.netlify.com/) or Render (https://render.com/)
     to deploy the website and publish
